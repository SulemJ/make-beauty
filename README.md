## make-beauty

This website uses "http://makeup-api.herokuapp.com/api/v1/products.json" API

## How it works

it accepts brand name and product type as a parameter;gives the product link, name, price and useful description about the product in a beautiful 

interface.

## Running the program

Clone this repo:

```
git clone https://github.com/SulemJ/make-beauty.git
cd make-beauty
```

## Running with node

make sure you have node.js installed

In the `make beauty` directory, install the dependencies using `npm install`.

Run the server: `nodemon index.js`.

Go to http://localhost:3000
